Руководство по использованию программы

Данная программа предназначена для работы с файлами формата XML.
В главном окне имеются два основных поля: левое предназначено для ввода, отображения и изменения XML тегов, правое окно служит для ирерхического отображения введённых тегов в виде дерева. Построения дерева тегов на основании введённой информаци происходит при нажатии кнопки "Построить дерево тегов", и дерево тегов будет оставаться неизменным до следующего нажатия данной кнопки.

В случае если необходимо получить информацию из файлов можно воспользоваться кнопкой "Считать", которая вызывает Проводник с возможностью открыть только файлы с расширением "*.xml". После открытия файла он вся информацию будет выведена в оба окна: правое окно представит информацию в виде дерева, а левое отобразит содержимое файла. После изменения введённой информации файл можно сохранить нажав кнопку "Запись".

Поле "Информационное окно" служит для отображение служебной инормации и ошибок, в случае их возникновения.
