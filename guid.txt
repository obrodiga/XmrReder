Руководство по использованию программы

Данная программа предназначена для работы с файлами формата XML.
В основном окне программы имеются два поля: в левое предназначено для ввода и изменения XML тегов, правое будет отображать дерево тегов на основании введённой информации. Для перестроения дерева тегов на основании введённой информации необходимо нажать конопку "Построить дерево".

В программе так же реализована возможность чтения информации из файлов с раширением "*.xml". Для этого необходимо нажать кнопку "Считать" в верхней части основного окна, а в открывшемсмя окне выбрать файл для чтения. В слечае если файл был изменён, его можно сохранить использую кнопку "Запись".

В случае если в работе возникли проблемы, информация будет выведена в поле "Информационное окно".
